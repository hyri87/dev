# Hyetoria — 사용자 매뉴얼

## 실행 방법

### 배포 exe 실행
```
Hyetoria.exe                    더블클릭으로 실행
Hyetoria.exe C:\path\file.dcm   파일을 바로 열면서 실행
```

### Windows 파일 연결 (더블클릭으로 열기)
1. `.dcm` 파일 우클릭 → **연결 프로그램** → **다른 앱 선택**
2. `Hyetoria.exe` 지정 → **항상 이 앱 사용** 체크
3. 이후 `.dcm` 파일 더블클릭 시 Hyetoria가 자동 실행

## 화면 구성

```
┌──────────────────────────────────────────────────────────────────┐
│  [파일 열기]  |  [Expand All]  [Collapse All]       상태 메시지  │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │
│                        태그 트리                                 │
│                                                                  │
├─────────────────────────────────┬────────────────────────────────┤
│                                 │                                │
│        헥스 덤프                │       이미지 뷰어              │
│                                 │                                │
└─────────────────────────────────┴────────────────────────────────┘
```

| 영역 | 설명 |
|------|------|
| 툴바 | 파일 열기 버튼, Expand/Collapse 버튼, 상태 메시지 |
| 태그 트리 | DICOM 태그 목록 (TAG, VR, Length, Name, Value, Offset) |
| 헥스 덤프 | 파일 원시 바이트를 16진수 + ASCII로 표시 |
| 이미지 뷰어 | DICOM 영상 표시 (그레이스케일 / 컬러 / 멀티프레임 / 비디오) |

> 각 영역 사이의 구분선을 마우스로 드래그하면 크기를 조절할 수 있습니다.

## 파일 열기

| 방법 | 조작 |
|------|------|
| 버튼 | 툴바의 **파일 열기** 클릭 → 파일 탐색기에서 선택 |
| 드래그 앤 드롭 | `.dcm` / `.ima` 파일을 Hyetoria 창 위에 드롭 |
| 커맨드라인 | `Hyetoria.exe <파일경로>` |
| 파일 연결 | 탐색기에서 `.dcm` 파일 더블클릭 (사전 연결 등록 필요) |

- 지원 확장자: `.dcm`, `.ima`, 모든 파일(`*.*`)
- 복수 파일 드롭 시 첫 번째 파일만 열림
- 파일 로드 후 타이틀바에 파일 경로, 상태바에 태그 수와 파일 크기 표시

## 태그 트리

### 기본 조작

| 조작 | 동작 |
|------|------|
| 행 클릭 | 해당 태그의 헥스 덤프 위치로 자동 스크롤 + 하이라이트 |
| **Expand All** 버튼 | 모든 Sequence 노드 펼치기 |
| **Collapse All** 버튼 | 모든 Sequence 노드 접기 |
| `▶` 아이콘 클릭 | 개별 Sequence 노드 펼치기/접기 |

### Value 복사

| 조작 | 동작 |
|------|------|
| `Ctrl+C` | 선택 행의 Value 값 클립보드 복사 |
| 우클릭 → **Value 복사** | 해당 행의 Value 값 클립보드 복사 |
| 우클릭 → **행 전체 복사** | TAG, VR, Length, Name, Value, Offset을 탭 구분으로 복사 (엑셀 붙여넣기 호환) |

### Value 팝업 (상세 보기)

| 조작 | 동작 |
|------|------|
| 행 **더블클릭** | Value 팝업창 열기 |
| 팝업 내 마우스 드래그 | 일부 텍스트 선택 |
| 팝업 내 **더블클릭** | 전체 텍스트 선택 |
| `Ctrl+C` | 선택한 텍스트 복사 (선택 없으면 전체 복사) |
| `Enter` 또는 `ESC` | 팝업 닫기 |

## 검색 (태그 트리)

| 단축키 | 동작 |
|--------|------|
| `Ctrl+F` | 검색 바 열기 |
| 텍스트 입력 | 실시간 검색 (TAG, VR, Name, Value 대상, 대소문자 무시) |
| `Enter` 또는 `▶` 버튼 | 다음 검색 결과로 이동 |
| `Shift+Enter` 또는 `◀` 버튼 | 이전 검색 결과로 이동 |
| `ESC` 또는 `✕` 버튼 | 검색 바 닫기 |

- 검색 결과 수 표시: `3 / 12` 형식
- 매치된 행: 노란색 배경
- 현재 포커스 행: 진한 오렌지 배경
- Sequence 안에 있는 결과는 부모 노드를 자동으로 펼쳐서 보여줌

## 헥스 덤프

- 파일 전체 바이트를 `XXXXXXXX  XX XX ... XX  ASCII` 형식으로 표시
- **Lazy Scroll**: 스크롤이 하단에 도달하면 자동으로 다음 데이터 로드
  - 파일 크기 제한 없음
- 태그 트리에서 행을 클릭하면 해당 offset으로 자동 스크롤되며 파란색 하이라이트 표시

## 이미지 뷰어

### 단일 프레임 영상
- WW/WL(Window Width / Window Level) 윈도잉 적용
- DICOM 태그에 WW/WL 값이 없으면 자동 계산
- 패널 크기 변경 시 영상이 자동으로 재렌더링

### 멀티프레임 영상

| 조작 | 동작 |
|------|------|
| `▶/⏸` 버튼 | 자동 재생 / 일시정지 |
| `◀` 버튼 | 이전 프레임 |
| `▶` 버튼 | 다음 프레임 |
| 슬라이더 드래그 | 원하는 프레임으로 이동 |
| 마우스 휠 | 프레임 탐색 (위: 이전, 아래: 다음) |

- 하단 Info bar에 `WW / WL`, `Frame X / N`, `FPS` 정보 표시

### 컬러 영상
- RGB/RGBA: 원본 색상 그대로 표시
- YBR 계열: 자동으로 RGB 변환 후 표시

### 비디오 DICOM
- MPEG2, MPEG4(H.264), HEVC(H.265) 자동 감지 및 재생
- 대용량 비디오도 프레임 1장씩 on-demand 로드 (메모리 절약)

### 이미지 없는 파일
- Pixel Data 없음: "No Image Data" 표시
- 디코딩 실패: 오류 메시지 + 로그 파일 경로 표시

## 전체 단축키 모음

| 단축키 | 동작 | 위치 |
|--------|------|------|
| `Ctrl+F` | 검색 바 열기 | 태그 트리 |
| `Ctrl+C` | Value 값 복사 | 태그 트리 |
| `Enter` | 다음 검색 결과 / Value 팝업 닫기 | 검색 바 / 팝업 |
| `Shift+Enter` | 이전 검색 결과 | 검색 바 |
| `ESC` | 검색 바 닫기 / Value 팝업 닫기 | 검색 바 / 팝업 |
| 더블클릭 | Value 팝업 열기 | 태그 트리 |
| 우클릭 | 컨텍스트 메뉴 (복사) | 태그 트리 |
| 마우스 휠 | 프레임 탐색 | 이미지 뷰어 |

## 오류 진단

비디오 DICOM 디코딩 실패 시 로그 파일이 자동 생성됩니다:
```
%TEMP%\hyetoria_error.log
```
이미지 뷰어에도 오류 메시지와 로그 파일 경로가 표시됩니다.
